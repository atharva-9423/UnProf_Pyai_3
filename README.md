# 📱 Contact Manager CLI

A lightweight Python contact manager with JSON storage and full error handling.

## ✨ Quick Features

**Add** • **Search** • **View** • **Persistent Storage** • **Error Logging**

## ▶️ Usage

```bash
python contact_manager.py
```

Pick an option from the menu and go. That's it.

## 🛠️ Built With

- **Python 3** — `json`, `logging`, custom exceptions
- **Day 2** — File handling & JSON persistence  
- **Day 3** — Error handling, logging, graceful recovery from corrupted files

## 🎯 Key Features

✅ Save contacts to `contacts.json`  
✅ Search by name, phone, or email (case-insensitive)  
✅ Never crashes — corrupted files are handled gracefully  
✅ Full activity log in `contacts.log`  
✅ Custom exceptions for clean error messages
